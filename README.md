# Регионы России

Справочник информации о регионах России, перечисленных [в ФИАС](https://fias.nalog.ru/):

- тип и название региона;
- федеральный округ;
- КЛАДР-код;
- ФИАС-код;
- код ОКАТО;
- код ОКТМО;
- код ИФНС;
- почтовый индекс;
- ISO-код;
- часовой пояс.

Данные, которых нет в ФИАС, получены из сервиса [DaData.ru](https://dadata.ru/)
